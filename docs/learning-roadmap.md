# Learning Roadmap

This document outlines my structured 180‑day learning roadmap covering Python, automation, data engineering, machine learning, deep learning, secure MLOps, AI security, AI governance, red teaming, and enterprise AI architecture.

## Phase 1 — Python Foundations (Days 1–30)
- Python basics, functions, data structures
- File handling, CLI tools
- Security scripting and automation
- Weekly projects and GitHub uploads

## Phase 2 — Data Engineering & ML (Days 31–60)
- Pandas, NumPy, ETL pipelines
- ML pipelines, evaluation, preprocessing
- Visualization dashboards
- Weekly ML systems

## Phase 3 — Deep Learning & Secure MLOps (Days 61–90)
- PyTorch, neural networks, CNNs, RNNs, Transformers
- Model registry, versioning, CI/CD
- Drift detection, monitoring, deployment

## Phase 4 — AI Security Engineering (Days 91–120)
- Adversarial attacks (FGSM, PGD, CW)
- Model poisoning & extraction
- Prompt injection defense
- AI supply chain security
- Red teaming frameworks

## Phase 5 — AI Governance Automation (Days 121–150)
- NIST AI RMF automation
- EU AI Act compliance scanning
- Risk scoring engines
- Governance dashboards
- Policy automation & reporting

## Phase 6 — Enterprise AI Security Architecture (Days 151–180)
- Architecture diagrams
- Threat models
- Governance integration
- Enterprise risk dashboards
- Final AI Security Blueprint

## Goal
This roadmap prepares me for advanced roles in AI Security Engineering, AI Governance, and AI Security Architecture, and supports my upcoming PhD in Artificial Intelligence.
